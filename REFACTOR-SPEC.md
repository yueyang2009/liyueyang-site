# 李岳阳财税顾问官网 - 重构规格书

## 总体要求
- 稳扎稳打，质量优先，可持续扩展
- SEO 友好，内容驱动，慢慢填充
- 深色极简风格（Linear风格）+ 金色点缀（保留现有品牌调性）
- 多页面架构（非单页），便于后续增加内容

## 页面结构

### 1. 首页 (index.html)
- Hero：slogan "让财税成为竞争力" + 头像/形象照占位
- 一句话定位：15年财税实战专家·注册会计师·律师
- 数据滚动条：15+年经验 / 500+服务企业 / 4位专家团队 / 7×24在线
- 核心服务概览（6大服务，点击进入服务页）
- 最新案例精选（3个，点击进入案例详情）
- 团队合影占位
- CTA：联系入口

### 2. 关于页 (about.html)
- 个人故事时间线（7年会所→8年CFO→创业）
- 四维资质展示（财务+税务+法律+经营）
- 证书展示区（可视化）
- 服务理念

### 3. 服务页 (services.html)
- 6大服务详细说明，每个服务独立卡片可展开
- 服务流程：首次咨询→上门尽调→方案设计→落地实施→长期跟踪
- 服务案例链接
- 价格/合作方式说明

### 4. 案例页 (cases.html) ← 重点！
- 案例列表（持续增加）
- 每个案例包含：行业标签、客户痛点、解决方案、效果数据
- 筛选功能（按行业/服务类型）
- 案例详情页模板

### 5. 博客页 (blog.html)
- 文章列表（按话题分类）
- 话题标签：社保夯实、两账合一、私户收款、股权设计、乱账梳理、财税合规、共享CFO、财税顾问、数智化改造、财务BP、财税赋能
- 文章详情页模板

### 6. 联系页 (contact.html)
- 飞书机器人二维码
- 服务区域
- 在线咨询表单（静态）
- 团队联系方式

## 技术要求
- 纯静态 HTML + CSS + JS，无框架依赖
- 每个页面独立 .html 文件
- 共享 CSS（style.css）和 JS（main.js）
- 响应式设计（Mobile First）
- 深色主题 #0a0a0a bg，金色 #c8a44e 点缀
- 字体：Noto Sans SC
- 动画：滚动渐入（保留现有效果）
- SEO：每个页面独立 title/description/og tags
- sitemap.xml 和 robots.txt

## 内容填充（首批）
1. 首页：公司slogan、数据、服务概览、案例精选
